Semantic HTML5 Elements

This is the <header> element - used for introductory content

Navigation

This is the <nav> element - used for navigation links

  • Main Content
  • Articles
  • Sidebar

Main Content Area

This is the <main> element - contains the primary content of the page

Section Element

The <section> element represents a section of a document, typically with a heading

Article Element

The <article> element represents a complete, independent piece of content

Articles can be:

  • Blog posts
  • News articles
  • Forum posts
  • Product reviews

Sidebar Content

The <aside> element contains content related to the main content but separate from it

Common uses:

  • Sidebars
  • Pull quotes
  • Related links
  • Advertisements

Footer

The <footer> element contains footer information for its nearest sectioning element
