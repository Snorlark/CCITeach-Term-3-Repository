Viewport width queries are the most common type of media query.

  • min-width: styles apply when viewport is wider than specified width
  • max-width: styles apply when viewport is narrower than specified width
  • width: styles apply at exact viewport width
@media (min-width: 768px) {
    /* Styles for tablets and up */
}

@media (max-width: 767px) {
    /* Styles for mobile devices */
}

@media (min-width: 768px) and (max-width: 1023px) {
    /* Styles for tablets only */
}

Orientation Queries

Try rotating your device to see the changes!

Portrait Mode Landscape Mode

Orientation queries help create layouts that work well in both portrait and landscape modes.

  • portrait: styles apply when height is greater than width
  • landscape: styles apply when width is greater than height
@media (orientation: portrait) {
    /* Styles for portrait mode */
}

@media (orientation: landscape) {
    /* Styles for landscape mode */
}

Resolution queries help optimize for different screen densities.

  • min-resolution: styles apply for higher resolution displays
  • max-resolution: styles apply for lower resolution displays
  • resolution: styles apply at exact resolution
@media (-webkit-min-device-pixel-ratio: 2),
       (min-resolution: 192dpi) {
    /* Styles for high-resolution displays */
}

Print media queries help optimize content for printing.

  • print: styles apply when printing
  • screen: styles apply for screen display
  • all: styles apply to all media types
@media print {
    /* Styles for printing */
    .screen-only {
        display: none;
    }
    body {
        font-size: 12pt;
    }
}

Try resizing your browser window and rotating your device to see the media queries in action!
