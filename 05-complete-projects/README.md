# Complete Projects

This directory contains full, working examples of web development projects that combine HTML, CSS, and responsive design principles.

## Contents

### 1. Personal Portfolio (`portfolio/`)
A responsive personal portfolio website that demonstrates:
- Modern, clean design
- Responsive navigation
- Project showcase
- Contact form
- About section
- Skills display
- Mobile-first approach

### 2. Restaurant Website (`restaurant/`)
A responsive restaurant website featuring:
- Menu display
- Reservation system
- Location and hours
- Photo gallery
- Contact information
- Responsive design for all devices

### 3. Blog Template (`blog/`)
A responsive blog template with:
- Article listings
- Category navigation
- Author profiles
- Comment system
- Search functionality
- Responsive typography

## How to Use

1. Each project is self-contained in its own directory
2. Projects include all necessary HTML, CSS, and assets
3. Follow the README in each project directory for specific setup instructions
4. Use these projects as templates or learning resources

## Project Structure

Each project directory contains:
- `index.html` - Main HTML file
- `css/` - Stylesheets
- `js/` - JavaScript files (if needed)
- `images/` - Image assets
- `README.md` - Project-specific documentation
